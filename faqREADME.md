# FAQ Chatbot

A simple FAQ chatbot built using Python and Tkinter.

## Features
- User-friendly GUI
- Keyword-based question matching
- Answers FAQs related to AI, Python, Machine Learning, and CodeAlpha
- Beginner-friendly implementation

## Technologies Used
- Python
- Tkinter

## Run the Project

```bash
python faq_chatbot.py
```
